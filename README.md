Nuristan is an AI-powered system designed to analyze food labels from packaged products. 
It extracts ingredient information using OCR and evaluates their health impact using a machine learning model. 
The system also provides personalized alerts based on user health conditions, allergies, and dietary preferences.


Features

📷 Extracts text from food labels using Tesseract OCR

🧹 Cleans and preprocesses ingredient data

🤖 Classifies ingredients using Random Forest ML model

⚠️ Detects harmful substances and allergens

👤 Provides personalized health warnings

⚡ Real-time and user-friendly analysis
